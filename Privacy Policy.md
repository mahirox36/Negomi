# Privacy Policy for Negomi<3

**Last Updated:** May 5, 2025

## Introduction

This Privacy Policy explains how Negomi<3 ("the Bot", "we", "us", or "our") collects, uses, and protects information obtained from users of our Discord bot service. We are committed to ensuring your privacy and protecting your data.

## 1. Information We Collect

### 1.1 Basic User Information

- Discord User IDs
- Current and previous usernames
- Account creation date
- Server join dates
- Profile pictures (for display purposes only)

### 1.2 Message Analytics

- Message counts and frequency
- Character and word counts
- Message content analysis for:
  - Emoji usage (both standard and custom)
  - Mentions of other users
  - Links shared and their domains
  - File attachments and their types (images, videos, audio, etc.)
  - Reply interactions
  - Message timestamps

### 1.3 Command Usage

- Commands used and their frequency
- Command execution context (channels, servers)
- Command parameters (non-sensitive)
- Command success/failure rates
- Command interaction patterns

### 1.4 User Interactions

- Message reply counts
- Reactions given and received
- User mentions (frequency and recipients)
- Channel participation patterns
- Voice channel usage statistics

### 1.5 Optional Information

- Birthday (if voluntarily provided by user)
- Server boost status
- Achievements and milestones
- Custom user preferences

## 2. How We Use Information

We use the collected information to:

- Provide and maintain the Bot's core functionality
- Generate user statistics and analytics displays
- Track command usage for feature optimization
- Create personalized user experiences and recommendations
- Monitor bot performance and usage patterns
- Improve bot features based on user interaction data
- Detect and prevent abuse or unauthorized usage
- Generate aggregated, anonymized insights

## 3. Information Sharing and Disclosure

- We do not sell, trade, or transfer your information to third parties
- Data is only shared within the server where it was collected
- Server administrators may see aggregated statistics but not individual message content
- Information may only be released if required by legal process or law enforcement
- Aggregate, anonymized statistics may be used for bot improvement and reporting
- We do not use your data for advertising or marketing purposes

## 4. Data Storage and Security

- All data is stored securely using industry-standard encryption
- We implement robust security measures to protect against unauthorized access
- Physical and electronic safeguards are in place to protect your information
- Access to data is strictly limited to bot functionality and authorized developers
- Data is retained only as long as necessary or until user requests deletion
- Regular backups are maintained in secure environments to prevent data loss
- We regularly review and update our security practices

## 5. User Rights

You have the right to:

- Access your collected data through bot commands
- Request a complete copy of your personal data
- Request deletion of your data
- Opt-out of specific types of data collection
- Restrict processing of your information
- Object to certain types of data processing
- Withdraw consent at any time
- Lodge a complaint with relevant supervisory authorities

### 5.1 How to Exercise Your Rights

You can exercise your rights by:

- Using specific bot commands designed for data management
- Contacting the bot owner via the information provided in Section 9
- Leaving servers where the Bot is present (for server-specific data)

## 6. Data Retention

- Active user data is retained while the bot remains in use
- Inactive user data may be automatically purged after 180 days of inactivity
- Users can request immediate data deletion at any time
- Some basic data may be retained for system functionality and security purposes
- Backup data is retained for a maximum of 30 days

## 7. Children's Privacy

The Bot does not knowingly collect data from users under 13 years of age (or higher minimum age as required by local laws). If we become aware that we have collected personal data from a child without parental consent, we will take steps to delete that information as quickly as possible.

## 8. Changes to Privacy Policy

- We reserve the right to update this privacy policy as needed
- Users will be notified of significant changes through:
  - Updated "Last Updated" date at the top of this policy
  - Server announcements (when possible)
- Continued use of the Bot after changes constitutes acceptance of the updated policy
- Previous versions of this policy will be archived and available upon request
